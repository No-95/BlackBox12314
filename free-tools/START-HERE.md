# Free Creator Stack for Agent Box

This folder contains a zero-cost starter toolkit for image and video generation.

## Included repos

1. ComfyUI
   - Main local workflow engine for image and video pipelines.
2. Fooocus
   - Easy image generator with good quality and simple prompting.
3. CogVideo
   - Open-source video generation codebase.
4. ComfyUI-Manager
   - Lets you manage ComfyUI custom nodes more easily.
5. ComfyUI-CogVideoXWrapper
   - Helps connect CogVideo-style workflows into ComfyUI.

## Best use order

1. Start with ComfyUI for flexible workflows.
2. Use Fooocus for fast image drafts.
3. Use CogVideo or ComfyUI video workflows for longer-term video generation.
4. Use CapCut free for final editing and polishing.

## Important note

These repos are free to use, but model weights are large and may still need to be downloaded separately depending on your machine and chosen workflow.

## Suggested next steps

1. Check your GPU and VRAM.
2. Install Python and Git if missing.
3. Launch ComfyUI first.
4. Connect it to the Content Creator Team later if desired.
