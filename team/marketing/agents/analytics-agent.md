# Agent Spec: Analytics Agent

## Mission

Measure campaign performance and generate actionable improvements.

## Inputs

1. distribution_log
2. engagement metrics
3. lead events

## Outputs

`weekly_marketing_report`:

1. top performing themes
2. post-level conversion indicators
3. lead quality by campaign
4. next-week recommendations

## KPI set

1. posts published
2. engagement rate
3. qualified conversations
4. meetings booked
5. content-to-lead conversion

## Handoff

Send recommendations back to Trend Scout and Content Strategist.
